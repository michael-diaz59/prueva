GITHUB

que es

github es un sistema de control de versiones que nos ayuda a controloar las distintas versiones de un programa se usa para proyectos complejos y o extensos, github es opensource creado por el mismo creador de linux, de hechos github fue creada para ayudarle a su creador en la construccion de un proyecto tan complejo como linux


usos

se usa para cordinar multiples desarrolladores en un proyecto, nos permite tener registros de todos los cambios y gracias a esto podemos navegar entre versiones para optimizar el codigo usando el mas eficiente git puede trabajar con repocitorios locales o remotos ademas de tener repositorios publicos y privados(con cuenta paga).


funciones basicas

ya que git interactua directamente con el sistema operativo se maneja con consola de comando(recomendado usar git bash).
bash es una consola con conceptos de unix.
en git hub hay 3 tipos de estado:"workin directory" donde se trabaja con los archivos.
"staging area" donde ponemos las versiones y archivos que estemos trabajando.
"repositorio" aqui es donde e guarda el ultimo cambio de una version y se guarda.


comandos basicos

git init nos permite crear un repositorio
git add nos permite pasar los trabajos del working directory al stating area
git status nos permite saber donde esta nuestro proyeco si en working,stading o un directorio y en cual
git commit nos permite pasarlo del stading area al repositorio osea crear la primera snapshops de nuestro proyecto
git push nos permite subir el proyecto a un repositorio remoto para que la comunidad o demas desarolladores puedan interactuar con este
git pull nos permite interactuar con los canvios al proyecto por parte de los demas desarolladores
git clone nos permite bajar una copia de un proyecto a la pc
