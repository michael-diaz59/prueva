<h1>que es github</h1>

github es un sistema de control de versiones que nos ayuda a controloar las distintas versiones de un programa se usa para proyectos complejos y o extensos. git es opensource creado por el mismo creador de linux. de hechos git fue creada para ayudarle a su creador en la construccion de un proyecto tan complejo como linux. se usa para cordinar multiples desarrolladores en un proyecto. nos permite tener registros de todos los cambios y gracias a esto podemos navegar entre versiones para optimizar el codigo usando el mas eficiente git puede trabajar con repocitorios locales o remotos ademas de tener repositorios publicos y privados(con cuenta paga).


<h2>funciones basicas</h2>

ya que git interactua directamente con el sistema operativo se maneja con consola de comando(recomendado usar git bash ) bash es una concola con conceptos de unix 3 tipos de estado workin directory_donde se trabaja con los archivos staging area donde ponemos las versiones y archivos que estemos trabajando repositorio aqui es donde e guarda el ultimo cambio de una version y se guarda
